Allure Beauty Atelier is a custom-built, headless e-commerce platform designed for the luxury beauty and fragrance market. Inspired by high-end apothecaries and editorial web design, this project merges minimalist aesthetics with advanced functionality. 

Beyond standard e-commerce features, Allure acts as a digital beauty consultant, utilizing AI to map user personalities to bespoke product routines, alongside a highly interactive digital gifting experience.

### âœ¨ Core Features

* **The Virtual Beauty Consultant (AI Personality Tester):** An interactive, fluid quiz built with Framer Motion that leverages the Vercel AI SDK. It analyzes user responses to generate personalized skincare and fragrance routines.
* **The Gift Curator:** A custom bundle-builder where users can visually drag-and-drop products into a digital gift box, add personalized notes, and generate shareable links for group gifting.
* **Headless Storefront:** A fully custom shopping experience with lightning-fast page loads, cart state management, and secure checkout via Stripe.
* **Persistent User Profiles:** Authentication and relational data architecture powered by Supabase to save curated gift boxes and past AI recommendations.
